Learning Reflection

"Fate leads the willing and drags along the reluctant."

Enrolling in the Web Systems and Technologies course helped me gain a deeper understanding of how websites are developed and maintained. Throughout the course, I learned how to create functional and responsive websites using different web development technologies. One of the most valuable aspects of this course was the opportunity to learn and use several web development languages, including HTML, CSS, JavaScript, and PHP. These languages helped me understand how both front-end and back-end development work together to create complete web applications. The knowledge and skills I gained from this course will also be useful in the development of our capstone project during my fourth year.

Although I am confident in my coding skills, I encountered several challenges throughout the course. The area I struggled with the most was website design using CSS. Unlike other development environments, such as NetBeans and Visual Studio Windows Forms, where user interface elements can be easily created through drag-and-drop tools, web design required me to manually write code for layouts and styling. I often had to refresh the browser repeatedly to see the effects of my modifications, which sometimes slowed down my workflow. Because I had become accustomed to drag-and-drop design tools, I initially found it difficult to create, arrange, and style web pages entirely through code.

To address these challenges, I explored different tools and strategies that improved my workflow. I installed extensions and plugins such as Live Server and Live Preview, which allowed me to view changes in real time without constantly refreshing the browser. These tools significantly improved my productivity and made the design process more efficient. In addition, I learned the importance of organizing my code properly by separating HTML, CSS, and JavaScript files and using clear naming conventions for classes and IDs. This made it easier to locate errors, maintain consistency, and debug design issues whenever they occurred.

Overall, this course provided me with valuable technical knowledge and practical experience in web development. It strengthened my understanding of how websites function and improved my ability to work with multiple programming languages. Through the challenges I faced and the solutions I implemented, I became a more capable and confident developer. The skills and lessons I gained from this course will be beneficial not only for future academic projects but also for my future career in the field of Information Technology.
